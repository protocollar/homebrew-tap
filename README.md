# Protocollar Homebrew Tap

Homebrew casks for [Protocollar](https://github.com/protocollar) projects.

## Install

```bash
brew tap protocollar/tap
brew install --cask fr8
```

## Available Casks

| Cask                                      | Description               |
|-------------------------------------------|---------------------------|
| [fr8](https://github.com/protocollar/fr8) | Terminal worktree manager |
